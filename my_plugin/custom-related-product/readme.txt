=== WooCommerce Product Bundles ===

WooCommerce custom related is a plugin helps you relate the product to each other by select box option!

== Description ==

WooCommerce custom related is a plugin helps you relate the product to each other by select box option!
The custom related has smart switching tool so when changing the selection of the WooCommerce custom related.


= Features =

- AJAX search & pick products
- Drag & drop to re-arrange
- Set title for each product
- Support variable product
- WPML integration (NEW)

= Translators =

Available Languages

- English (Default)
- German (Thanks to Rosemarie Braun)


== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WooCommerce Product Bundles", Install & Activate it
4. Now when you create/edit a product you can choose product type is "custom-related-product", then add many products as you want
5. Almost done! click Save and see the result
